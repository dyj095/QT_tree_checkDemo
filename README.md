# QT_tree_checkDemo
## 一、简介：<br>
基于QTreeWidget实现的带Checkbox多级组织结构选择树

## 二、运行：<br>
注：运行本Demo需要修改mainwindow.cpp第228行代码中的images目录的绝对路径；

开发环境：<br>
1、QT版本：Qt5.11.0<br>
2、编译器：MinGW<br>
